Getting plans into the shop

Print plans

Use Export → Print plans (or ⌘P / Ctrl+P) for a one-page drawing sheet with cut list, elevations, and title block — the fastest path to the bench.

Build mode on your phone

Tap Build for board-by-board cutting diagrams and a checkable assembly list. The screen stays awake while you work. Share a BB4: code from Export → Share if you designed on a desktop and build on a phone.

Cut list spreadsheet (.csv)

Export Cut list (.csv) for a spreadsheet or shopping list app. Lengths already include joinery allowances; dimensions use your display units plus a millimetre column.

Drawing sheet (.svg)

Export Drawing sheet (.svg) for a vector file you can print, mark up, or open in any SVG viewer — same sheet as Print.

3D model / AR (.glb)

The .glb export is the universal 3D format. On an Android phone, opening the file offers View in your space — the piece stands in your room at true size. On desktop, any glTF viewer reads it. On iOS, run it through Apple’s free Reality Converter for AR Quick Look. Geometry is exact millimetres.

SketchUp mesh (.dae)

  1. Export SketchUp mesh (.dae) from the Export menu.
  2. In SketchUp choose File > Import… and pick COLLADA (*.dae) as the file type.
  3. Select the downloaded file and click Import — parts arrive named, at exact size, right side up.

SketchUp build script (.rb)

The Ruby script rebuilds the model natively: one component per unique part, instances placed exactly, a tag per role, all in one undo step. Open Window > Ruby Console, paste the script, press Enter.

Claude + SketchUp connector

Working inside Claude with the Trimble SketchUp connector? Export your DesignSpec (.json), paste it into the conversation, and ask Claude to build the piece in your live SketchUp model.
