Getting your design into SketchUp

Mesh import (.dae)

  1. Export SketchUp mesh (.dae) from the Export menu.
  2. In SketchUp choose File > Import… and pick COLLADA (*.dae) as the file type.
  3. Select the downloaded file and click Import — parts arrive named, at exact size, right side up.

Build script (.rb) — the parametric way

The Ruby script rebuilds the model natively: one component per unique part, instances placed exactly, a tag per role, all in one undo step. Open Window > Ruby Console, paste the script, press Enter. (Or drop the file into your Plugins folder.) Serious users want this one — components stay editable.

3D model / AR (.glb)

The .glb export is the universal 3D format. On an Android phone, opening the file offers View in your space — the piece stands in your room at true size. On desktop, any glTF viewer (or the three.js editor) reads it. On iOS, run it through Apple’s free Reality Converter to get an AR Quick Look file. Geometry is exact millimetres, rotations included.

Integrations: SketchUp connector in Claude

Working inside Claude with the Trimble SketchUp connector? Export your DesignSpec (.json), paste it into the conversation, and ask Claude to build the piece directly in your live SketchUp model. The spec carries every dimension, so nothing is lost in translation.
